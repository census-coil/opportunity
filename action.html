Participate in an upcoming sprint cohort

The U.S. Census Bureau connects tech developers, local leaders, and non-profits with problem statements on issues like jobs, transportation, and housing, and facilitates
user-centered tech development sprints to create new digital solutions. Links to Sprints page.




Run your own sprint

Use this process to run your own sprint. Local leaders, technologists, students, or anyone interested in building data-driven solutions can start a project. View The Opportunity Project toolkit, a step-by-step guide to the process used by the Census Bureau team, and adapt it for your own project. Links to toolkit.
